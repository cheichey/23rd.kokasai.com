# 23rd.kokasai.com

## 開発

### コマンド

#### 依存関係の導入
```shell
yarn install
```

#### テストサーバーの起動
```shell
yarn start
```

#### ビルド
```shell
yarn build
```

#### 依存関係の更新確認
```shell
npx -p npm-check-updates  -c "ncu"
```

#### 依存関係の更新
```shell
npx -p npm-check-updates  -c "ncu -u"
```

#### package.json の整理
```shell
npx sort-package-json
```
